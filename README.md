# Dungeon Dash!
Modified version of the original [Dungeon Dash](https://github.com/mipearson/dungeondash).

## Add ons:
* Score system
* Lives system
* Added potions - Modifies FOV
* Added Chests/Coins - Adds to the score system
* Added Scenes: Start and Game Over screens.
